# PHP To-Do App (Pure PHP)

A simple To-Do application built with Pure PHP, PostgreSQL, and Docker.

## Features
- User Registration & Login
- Session-based Authentication
- CRUD To-Do
- PostgreSQL (PDO)
- Docker Compose

## Tech Stack
- PHP 8.2
- Apache
- PostgreSQL 15
- Docker

## Setup
```bash
docker compose up -d
